Guide

Using Life OS effectively

Tailored • Editable • Real-time
  1. Complete onboarding: pick modules you want and set your tracking preferences.
  2. Dashboard: skim your top cards, then click into any module for detail views.
  3. Tasks/Projects: quick-add, edit, complete, or remove tasks directly on the Kanban columns.
  4. Events: add/mark done/edit/remove right from the Events page.
  5. Reminders: capture, complete, or edit due dates inline.
  6. Notes: add, edit, tag, or delete in place; keep ideas organized.
  7. Tailor Profile: adjust dashboard visibility and module settings anytime.
Tips
  • Use “Remember this browser” at login for longer sessions.
  • Modules you disable stay hidden from nav and dashboard.
  • API writes: set LIFEOS_TOKEN and send X-Auth-Token to use /api endpoints.
